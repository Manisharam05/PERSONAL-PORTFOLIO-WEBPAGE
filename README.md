This is a personal developer portfolio website designed to showcase professional skills, projects, and contact information. It has a clean, modern, and responsive layout with sections such as Home, About, Projects, and Contact. The website includes smooth navigation, a hero section with a call-to-action, project cards displaying technologies used, and a contact form for user interaction.

Languages & Technologies Used:

HTML5 – for structuring the content and layout of the website

CSS3 – for styling, layout design, responsiveness, and animations

JavaScript – for basic interactivity such as the mobile navigation (hamburger menu)

Google Fonts (Roboto) – for modern typography

Overall, this website is a static, responsive portfolio site ideal for presenting a developer’s profile and projects in a professional manner.
